# UwUify

Adds a new UwU language option.

## Configuration

-   **Enable Dialogue UwU**:
    -   Apply UwU transformation to dialogue text.
    -   **Default**: `Enabled`
-   **Enable UI UwU**:
    -   Apply UwU transformation to UI and HUD.
    -   **Default**: `Enabled`
-   **Enable Tooltips UwU**:
    -   Apply UwU transformation to tooltips.
    -   **Default**: `Enabled`
-   **Enable Equipment UwU**:
    -   Apply UwU transformation to equipment names and descriptions.
    -   **Default**: `Enabled`
-   **Enable Ships UwU**:
    -   Apply UwU transformation to ship names and descriptions.
    -   **Default**: `Enabled`
-   **Enable World UwU**:
    -   Apply UwU transformation to lore, oddities, and services.
    -   **Default**: `Enabled`
-   **Enable Tips UwU**:
    -   Apply UwU transformation to loading screen tips.
    -   **Default**: `Enabled`
-   **Enable Tunes UwU**:
    -   Apply UwU transformation to ship tuning descriptions.
    -   **Default**: `Enabled`

> [!WARNING]
> Requires HevLib to function
> https://github.com/rwqfsfasxc100/HevLib/releases
